Berkshire Hathaway

CMST 308 User Experience and Interface Design

Overview

This project redesigned the Berkshire Hathaway website to enhance its usability and user experience. The existing site, characterized by a simple, text-heavy interface and stock HTML design, lacked key design elements and modern functionality. My role involved conducting user research, creating user personas, analyzing user needs and tasks, and identifying usability requirements. The redesign aimed to create a more accessible, intuitive, and visually appealing platform for users seeking information about Berkshire Hathaway.

Goal

The primary goal of the redesign was to maintain the site's core objective of providing direct access to the company’s documents while improving its overall user-friendliness.
